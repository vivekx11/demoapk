# DemoAPK Repository

## Description
This repository contains 12 important files/folders copied from the apk1.demo project.

## Important Files and Folders

1. **app/** - Main application code and source files
2. **.vscode/** - VS Code configuration and settings
3. **data/** - Data files and resources
4. **docs/** - Documentation files
5. **domain/** - Domain models and business logic
6. **gradle/** - Gradle configuration files
7. **ml/** - Machine learning models and resources
8. **scripts/** - Build and deployment scripts
9. **build.gradle.kts** - Gradle build configuration
10. **firebase-test-lab-config.yml** - Firebase testing configuration
11. **gradle.properties** - Gradle properties configuration
12. **proguard-rules.pro** - ProGuard obfuscation rules

## Source Repository
These files were copied from: [apk1.demo](https://github.com/vivekx11/apk1.demo)

## Structure
This project serves as a reference copy of important configuration and source files from the main apk1.demo project.
